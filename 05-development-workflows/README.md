### Modules:
Contains the required resources for setting up the infrastructure

#### Environments:
Contains different configurations of each environment. The values are defined under ""```"".tfvars```

#### Global:
The global modules defines different sorts of permissions for different IAM roles, state bucket and logs.

#### Scripts:
The scripts consist bash script for the deployment of infrastructure based on the command passed by user in the CLI.

#### Execution flow:
```bash
./deploy.sh
```

The bash script does:
```bash
cd environments/dev
terraform apply
```

Terraform loads: Root module (dev) -> VPC module -> EC2 module -> RDS module
